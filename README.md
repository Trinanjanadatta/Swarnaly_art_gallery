# Artist Portfolio Website 🎨

A responsive artist portfolio website designed to showcase an artist's work through a clean, immersive, and visually engaging experience. The website provides a dedicated space for presenting artwork, sharing information about the artist, and creating a professional online presence.

## 🌐 Live Demo

🔗 [https://your-portfolio-link.com](https://swarnaly-art-gallery.vercel.app/)

## ✨ Features

* Responsive design for desktop, tablet, and mobile devices
* Artwork gallery with modern layouts
* About the Artist section
* Smooth hover effects and interactive elements
* CSS animations and transitions
* Custom layouts built with CSS Grid
* Clean and intuitive navigation
* Optimized user experience across different screen sizes

## 🛠️ Technologies Used

* HTML5
* CSS3

## 📚 Concepts Implemented

* CSS Grid for non-uniform layouts
* Responsive design using media queries
* CSS animations and transitions
* CSS transformations and hover effects
* Modern layout and styling techniques
